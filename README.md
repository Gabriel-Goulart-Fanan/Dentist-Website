🦷 Dentista — Site Odontológico
Projeto de site institucional para consultório odontológico, desenvolvido com a técnica Mobile First e adaptado para desktop.

📋 Sobre o Projeto
Site criado para uma clínica odontológica fictícia, usando como referência e inspiração o trabalho de uma dentista real. 
O objetivo foi praticar a construção de uma landing page profissional e responsiva, partindo do layout mobile e expandindo para telas maiores.
O projeto foi desenvolvido durante estudos e aulas acompanhando o canal da Larissa Kich, com foco em boas práticas de HTML, CSS e responsividade.

✨ Funcionalidades
Layout responsivo (Mobile First → Desktop)
Navbar com links de navegação e botão de contato em destaque
Hero section com headline impactante e chamadas para ação
Design limpo com paleta em tons de teal/verde-água e branco
Elementos decorativos (dentes 3D, folhas, ícones) para identidade visual


🛠️ Tecnologias Utilizadas

HTML5 — estrutura semântica
CSS3 — estilização, Flexbox/Grid e media queries
Mobile First — desenvolvimento iniciado pela versão mobile
